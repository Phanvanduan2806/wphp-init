Private themes and plugins manager for WordPress.

== Description ==

Display themes and plugins list from private repo.

Install / Reinstall / Update theme or plugin.

Import config from file: LiteSpeed Cache

== Installation ==

1. Upload `wp-site-manager` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
